Create an eggroll table feed with data specified by parameters provided

Parameters:
  • ids – 1D numpy array
  • X – 2D numpy array
  • y – 2D numpy array
  • sample_range – a tuple specifies the range of samples to feed into dtable
  • feature_range – a tuple specifies the range of features to feed into dtable
  • tables_name – a dictionary specifies table namespace (with key table_ns) and table name (with key table_name)
  • partition – number of partition used when creating the dtable
Returns:

an eggroll dtable
